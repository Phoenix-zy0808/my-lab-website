# 📐 页面布局参考图 (Reference Designs)

本目录存放页面布局的参考图片，用于指导开发。

---

## 📁 文件命名规范

| 文件名 | 对应页面 | 说明 |
|--------|---------|------|
| `01-homepage.jpg` | 首页 (Home) | Hero 大图 + 核心入口卡片 + 最新新闻 |
| `02-people-page.jpg` | 团队成员 (People) | PI、研究生、毕业生分组展示 |
| `03-research-page.jpg` | 研究方向 (Research) | 拉曼光谱、红外光谱等课题介绍 |
| `04-publications.jpg` | 成果展示 (Publications) | 论文、专利、软著列表 |
| `05-facilities.jpg` | 仪器设备 (Facilities) | 设备参数、图片、使用指南 |
| `06-gallery.jpg` | 课题组生活 (Gallery) | 团建、会议、花絮画廊 |
| `07-news.jpg` | 研究进展 (News) | 时间轴形式的动态列表 |
| `08-contact.jpg` | 联系我们 (Contact) | 地图、邮箱、电话、社交媒体 |

---

## 📷 参考图内容建议

每张参考图应尽量清晰展示：
- [ ] **整体布局**（几列、间距、对齐方式）
- [ ] **卡片样式**（阴影、圆角、悬停效果）
- [ ] **颜色搭配**（主色、背景色、文字颜色）
- [ ] **字体层级**（标题大小、正文字号）
- [ ] **图片位置**（Hero 图、封面图、缩略图）

---

## 📝 使用说明

1. 将参考图片放入本目录
2. 按上述规范命名文件
3. 开发者会根据图片分析并实现对应布局
