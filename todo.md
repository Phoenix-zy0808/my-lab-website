# 🧬 智能谱学与分子感知实验室官网 - 前端开发详细规划

## 🎨 1. UI 视觉与全局样式规范
**状态：** ⏳ 待配置 (Tailwind CSS)

* **主色调 (Primary)：** 深海蓝 `#0a3a52` 或 `blue-900`。用于网页顶部的透明导航栏 hover 效果、页脚 (Footer) 背景、按钮以及重要标题文字。体现科研的沉稳感。
* **辅色调 (Secondary)：** 灰青色 `#578b9e`。用于次要图标（如首页三个圆圈 icon 的背景）、卡片悬浮边缘。
* **背景色 (Background)：** 网页主背景使用极浅的灰 `#f8f9fa`，所有内容卡片使用纯白 `#ffffff` 并增加微阴影 `shadow-md`，以凸显层级。
* **页面版心宽度：** 固定主内容区最大宽度 `max-w-[1200px]`，居中显示 `mx-auto`，不考虑移动端自适应。

## 🖥️ 3. 页面级详细设计与开发列表

### 🟢 3.1 首页 (`/`)
* **顶部导航 (Header)：** 绝对定位在最顶端。左侧为实验室 Logo 与中英文名称；右侧为 8 个横排导航链接；文字为白色，背景全透明。
* **Hero 视觉区：** 高度约 `60vh`，使用细胞/DNA结构高清大图作为 `background-image`，叠加一层 `bg-black/30` 的黑色半透明遮罩让文字更清晰。正中央放巨大的中英文实验室名称。
* **悬浮导航卡片 (参考图1)：** 在 Hero 区域底部使用负边距（如 `-mt-20`）向上悬浮三个卡片：
    * [课题组简介]、[课题组成员]、[研究方向]。
    * 卡片顶部带有灰青色圆形 Icon，卡片内放图，底部带有半透明文字遮罩。
* **公共页脚 (Footer)：** 深海蓝背景，居中排版。包含中国计量大学实验室具体地址、联系邮箱 (Email) 以及友情链接。

### 🟢 3.2 课题组简介页 (`/about`)
* **简版 Hero 头部：** 高度约 `30vh` 的窄版背景大图，中央文字为“课题组简介”。
* **主内容卡片 (参考图2)：** * 外部包裹一个纯白底色的宽卡片。左上角带有竖线修饰的标题“| 课题组简介”。
    * 上半部分：标准 Markdown 排版式的文字介绍（支持首行缩进，行高设定为 `leading-relaxed` 以提升阅读体验）。
    * 下半部分：居中展示对比实验大图（如 DMSO / Apoptosis 细胞四宫格显微图），底部附带邮箱联系方式，呼吁学生加入。

### 🟢 3.3 课题组成员页 (`/team`)
* **人员分类 Tabs：** 卡片顶部使用横排 Flex 布局实现切换按钮（PI介绍 / 副研究员 / 科研助理 / 博士后 等），点击动态高亮当前项。
* **人员详情卡片 (参考图3)：** 采用左右分栏布局。
    * **左侧：** 固定宽度 `w-48`，展示高精度的个人职业照。
    * **右侧：** 垂直流式排版，按顺序渲染：个人资料（姓名/Email） -> 个人简介（大段文字） -> 教育与工作简历（时间轴或简单列表） -> 研究兴趣（有序列表） -> 代表性论文（带上下标的富文本列表）。

### 🟢 3.4 成果展示页 (`/achievements`)
* **分类 Tabs：** 顶部展示 [软著]、[专利]、[论文]、[学科竞赛] 切换栏。
* **成果列表：** 对应分类下，循环渲染列表条目。每个条目包含：成果名称、年份、作者/成员、一个显眼的“查看 PDF”按钮。
* **沉浸式 PDF 弹窗组件 (`<PdfModal />`)：**
    * 全局隐藏，点击“查看 PDF”时弹出。
    * `z-index: 50` 的全屏黑色半透明遮罩。
    * 屏幕正中央展示 `width: 80%; height: 90vh;` 的 `iframe` 标签，直接加载 `public/pdfs/...` 文件。右上角提供 `[X] 关闭` 按钮。
